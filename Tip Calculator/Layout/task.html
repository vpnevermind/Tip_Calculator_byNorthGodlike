Description

Normally, the amount of tip is calculated as a percentage of the bill. So, our app will need to know the amount due and the desired tip percentage to do the calculations. In this stage, we will add the corresponding layout fields for the user to interact with.

The user should be able to input the bill value by hand, regulate the tip percentage with a slider, and see the calculation results in a text field.

Objectives

Add the following components to the application layout:

  1. An EditText where users enter the bill value with the id attribute edit_text , the inputType attribute numberDecimal and hint attribute "Insert Bill Value";

  2. A SeekBar widget for specifying the tip percentage with the id attribute seek_bar. Set the max attribute to 100 and the progress attribute to 15;

  3. A TextView with id bill_value_tv that will be used to display the bill value. Keep it empty at this stage;

  4. A TextView with id tip_percent_tv that will be used to display the tip percent. Keep it empty at this stage;

  5. A TextView with id tip_amount_tv that will be used to display the calculation result. Keep it empty at this stage;
